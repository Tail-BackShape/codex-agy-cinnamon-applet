# Changelog

## 0.1.0 - 2026-07-15

- Add independent Codex and Antigravity refreshes through the CodexBar Linux CLI.
- Add schema-tolerant quota normalization, unknown usage handling, deduplication, and worst-window summaries.
- Add stale last-known-good state, per-provider errors, manual refresh, and configurable used/remaining display.
- Add installer, uninstaller, privacy-preserving diagnostics, captured fixtures, tests, and documentation.
